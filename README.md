# My CV Page

A personal CV web page built with semantic HTML5.

## Description

This is a simple CV page built as part of my HTML learning journey.
It displays my personal information, skills, education, and social links.

## Features

- Semantic HTML5 structure (`header`, `main`, `section`)
- Proper use of `<abbr>`, `<time>`, and `<address>` tags


## Technologies Used

- HTML5
- CSS (inline `<style>`)
